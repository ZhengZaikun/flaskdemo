About This Wikipedia Search App

This is a demonstration project built using the Python Flask framework and the Wikipedia API.

It showcases the fundamentals of Web programming learned in the CP1404 course, including: routing, view functions, and the use of Jinja templating.

Key technologies used:

  • Python Flask Framework
  • Jinja2 Template Inheritance
  • The Wikipedia Python library
